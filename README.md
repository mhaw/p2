# Mike Haw's P2

## Live URL
Live URL <http://p2.mikehaw.me>

## Description
Simple password generator based on xkcd's password comic. 

## Demo
+ Demo <http://screencast.com/t/dvpPRDRN4>

## Details
#####Options include:

+ Select 1-20 words.
+ Add a symbol.
+ Add a number.
+ Make password uppercase.
+ Select separator type.
